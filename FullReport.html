2.1 Notifications

h = Slab thickness (mm)
fctm = Mean Axial Tensile Strengh (MPa)
fctd = Flexural Strengh(MPa)
fr1 = The residual flexural strength at CMOD 0.5 (MPa)
fr4 = The residual flexural strength at CMOD 3.5 (MPa)
σf1 = Mean axial tensile strength fr1 (MPa)
σf4 = Mean axial tensile strength fr4 (MPa)
pdw = Pile distance width (mm)
Weff = Effective Bearing Width (mm)
b1 = max(slab thickness/2, 150) (mm)
L1 = longitudinal Center span (m)
L2 = Leteral Center span (m)
L3 = longitudinal edge span (m)
L4 = Leteral edge span (m)
Leff1 = Effective longitudinal Center span (m)
Leff2 = Effective Leteral Center span (m)
Leff3 = Effective longitudinal edge span (m)
Leff4 = Effective Leteral edge span (m)
Ed = Effective Depth (mm)
Fsf = Safe factor for concrete with fiber
Mutl = Residual Moment (kNm/m)
fy = steel rebar yeild strengh (MPa)
fyd = Fractored steel rebar yeild strengh (MPa)
d1 = Cover Depth (mm)
ds = rebar Depth (mm)
dis = rebar distance (mm)
dia = rebar diameter (mm)
EA = Rebar quantity (EA)
As = Rebar amount (mm^2)
Cden = Concrete Density(kg/m3)
Fself = Safe factor for dead load
Fdn = Safe factor for dynamicload
Selfweight = slab dead load (kN/m2)
Fself = Factored selfweight (kN/m2)
Rack = Single racking Load (kN)
Frack = Factored racking Load (kN)
MHE = Single wheel Load(kN)
Fmhe = Factored MHE Load (kN)
UDL = Uniform Load (kN)
Fudl = Factored UDL load (kN)
Fudlt = Factored (UDL + Selfweight) (kN)
LL = Line load (kN)
PL = Point load (kN)
Fld = Factored line load (kN)
Fpl = Factored piont load (kN)
Rfan = distance of Closest pile to point load (m)
F = Moment capacity reduction factor
